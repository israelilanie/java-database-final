# Portfolio Notes

This project demonstrates a full-stack backend architecture used by warehouse admins to manage retail stores.

## Highlights
- **Spring Boot + Hibernate** for structured inventory, orders, and store management.
- **MongoDB** for unstructured reviews.
- **REST endpoints** consumed by the admin UI.
- **Docker Compose** for a reproducible full-stack environment.
- **Stored procedure examples** for analytics workflows.

## Demo Checklist
- Add a store and validate the ID.
- Create products and link them to store inventory.
- Filter and search inventory by category/name.
- Place an order and verify inventory decrement.
- View customer reviews for a product.
