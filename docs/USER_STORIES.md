# User Stories (Admin)

## Store Management
- As an admin, I can create and maintain store records with names and addresses so the warehouse can track store locations.
- As an admin, I can validate a store ID before placing orders so that orders are always associated with a valid store.

## Product Catalog
- As an admin, I can add, edit, and delete products so the catalog stays accurate.
- As an admin, I can search products by name and filter by category so I can quickly locate items.
- As an admin, I can ensure SKU uniqueness to prevent duplicates.

## Inventory Management
- As an admin, I can add products to store inventory with stock levels so each store has the correct quantities.
- As an admin, I can update stock levels as inventory changes.
- As an admin, I can filter inventory by store, product name, or category to understand stock distribution.
- As an admin, I can validate stock before placing an order to avoid overselling.

## Order Placement
- As an admin, I can place orders on behalf of customers (phone/in-store requests) with customer details and order items.
- As an admin, I can see orders reduce inventory quantities automatically.

## Reviews (MongoDB)
- As an admin, I can view customer reviews by store and product to understand product feedback.

## Reporting & Analytics (Stored Procedures)
- As an admin, I can run SQL stored procedures to view monthly sales by store, company-wide sales, and top-selling products by category.
